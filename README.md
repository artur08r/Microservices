📌 Описание проекта

Разработал демо-систему на основе микросервисной архитектуры, реализующую асинхронное и синхронное взаимодействие сервисов через Kafka и REST API. Система обрабатывает входящие события, распределяет их между сервисами, сохраняет данные в различные хранилища (MongoDB, Redis, PostgreSQL) в зависимости от бизнес-логики и публикует результаты обработки обратно в Kafka.

Проект развернут в контейнерной среде с использованием Docker Compose, что обеспечивает простоту локального запуска и тестирования всей инфраструктуры.

⚙️ Стек технологий
Java, Spring Boot
Apache Kafka, Kafka UI (Redpanda Console)
REST API (Spring MVC)
PostgreSQL + Liquibase
MongoDB
Redis
Docker, Docker Compose
Gradle
SLF4J (логирование)
